# 🔌 SPI Simulator

เครื่องมือจำลองการทำงาน **SPI (Serial Peripheral Interface) Protocol** สำหรับนักเรียนระดับ ปวส.

## 🎯 วัตถุประสงค์

- ช่วยให้นักเรียนเข้าใจการทำงานของ SPI Protocol ได้อย่างง่ายดาย
- จำลองการส่งข้อมูลแบบ real-time พร้อม animation
- ทดสอบความเข้าใจผ่านแบบทดสอบ interactive

## ✨ ฟีเจอร์หลัก

### 📚 โหมดเรียนรู้ (Learning Mode)

- **Visualization แบบ Real-time**: แสดงการเชื่อมต่อระหว่าง Master และ Slave พร้อมสัญญาณ 4 เส้น
- **Oscilloscope View**: แสดง waveform ของสัญญาณแบบ real-time
- **Animation Step-by-Step**: ดูการทำงานทีละขั้นตอนพร้อมคำอธิบาย
- **รองรับ SPI Modes**: Mode 0 และ Mode 3
- **ตัวอย่างโค้ด Arduino**: Master และ Slave code

### 📝 โหมดทดสอบ (Practice Mode)

แบบทดสอบ 10 ข้อ ครอบคลุมเนื้อหา SPI ทั้งหมด

## 🚀 วิธีใช้งาน

เปิดไฟล์ `index.html` ด้วยเว็บบราวเซอร์ หรือใช้ Live Server

## 📖 เนื้อหาที่สอน

1. ส่วนประกอบพื้นฐาน SPI (MOSI, MISO, SCLK, SS)
2. การทำงานของ Master/Slave
3. SPI Modes (CPOL/CPHA)
4. การสื่อสาร Full-Duplex
5. ตัวอย่างโค้ด Arduino

## 🛠️ เทคโนโลยี

- HTML5, CSS3, Vanilla JavaScript
- Canvas API สำหรับ oscilloscope
