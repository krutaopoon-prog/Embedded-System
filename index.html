🔧 Embedded System Simulators

เครื่องมือจำลองการทำงานโปรโตคอลต่างๆ ในระบบ Embedded สำหรับนักเรียนระดับ ปวส.

🎓 ประกาศนียบัตรวิชาชีพชั้นสูง
🔌

SPI Protocol

✅ พร้อมใช้งาน

จำลองการทำงานของ Serial Peripheral Interface แบบ real-time พร้อม oscilloscope และแบบทดสอบ 10 ข้อ

เข้าชม Simulator →
🔗

UART Protocol

🚧 อยู่ระหว่างพัฒนา

จำลองการทำงานของ Universal Asynchronous Receiver-Transmitter สำหรับการสื่อสารแบบ asynchronous

🔄

I2C Protocol

🚧 อยู่ระหว่างพัฒนา

จำลองการทำงานของ Inter-Integrated Circuit สำหรับการเชื่อมต่อหลายอุปกรณ์ด้วยสาย 2 เส้น

🚌

CAN Bus

🚧 อยู่ระหว่างพัฒนา

จำลองการทำงานของ Controller Area Network สำหรับระบบยานยนต์และอุตสาหกรรม

🌡️

One-Wire

🚧 อยู่ระหว่างพัฒนา

จำลองการทำงานของ Dallas Semiconductor One-Wire Protocol สำหรับเซนเซอร์อุณหภูมิ DS18B20

✨ ฟีเจอร์หลักของ Simulators

� รองรับทุกอุปกรณ์ Desktop, Tablet, Mobile
🌐 ไม่ต้องติดตั้ง ใช้งานผ่านเว็บบราวเซอร์ได้เลย

🔧 ฟีเจอร์เสริม (ตามแต่ละโปรโตคอล)
